# 미루기의 끝

깔끔한 개발 블로그는 언제나 하고 싶었다. 다른 사람들의 글에 도움을 받아오며 살아왔으면 그래도 조금은 기여를 해야하지 않을까 싶었다. 하지만 귀차니즘과 두려움, 핑계로 일관하며 하지 않고 있었다.



문득  '이런 태도로 가면 평생 개발자 하고싶은 마음은 접어야 겠다.' 는 생각이 들었다. 태도를 접던가, 개발자를 일찍 접던가 둘 중 하나를 접어야 할 것 같았다.



뭐.... 그렇다. 계속 게으름 떨면 평생 남들만 부러워하며 살 것 같았다.
